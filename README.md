# Проект "Красивые места"
## одностраничный сайт с возможностью добавлять и удалять фотографии, ставить им «лайк»

[Проектная работа на GitHub Pages](https://semvas.github.io/mesto/index.html)

Результат с четвертой по девятую практических работ на курсе веб-разработки [Практикума](https://praktikum.yandex.ru/web/). Используются:

* API
* ООП в интерфейсах
* модули в JavaScript
* базовый JavaScript и работа с DOM элементами
* разметка и вёрстка форм
* валидация форм с JavaScript
* инструменты адаптивной верстки
* грид-лейаут
* флекс-верстка
* позиционирование
* методология БЭМ
* файловая структура Nested BEM
* настройка и сборка Вебпаком
